# Changelog

### Added

- Был добален переключатель на светлую тему в настройки
- Добавлена поддержка кода через Apple ID

### Changed

- Переработан внешний вид карточек задач, теперь они с закругленными углами

### Deprecated

- Старый метод экспорта в XML теперь не актуален, планируем полностью удалить его в версии 2.0.0

### Removed

- Удален старый модуль интеграции с Twitter, так как мы отключили эту функцию месяц назад

### Fixed

- Исправлена ошибка, из-за которой приложение закрывалось при вводе специальных символов в поле email

### Security

- Обновлена версия библиотеки шифрования для закрытия критической уязвимости CVE-2026-0199
